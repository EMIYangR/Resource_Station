#### 部分FAQ

##### Q：为什么访问/下载速度那么慢？

A：Onedrive不会限制用户的访问/下载速度，您的速度过慢可能受以下因素影响：

​	1、网速； 

​	2、与Microsoft服务器响应不及时或未响应。

##### Q：为什么下载文件后显示损坏？

A：由于微软方面原因，下载本站点文件时不能断开下载。

##### Q：如何验证本此站点提供的文件本身没有损坏？

A：在自述文件附带了部分文件的CRC32值，您可以自行使用计算后进行比对。

##### Q：为什么有些文件夹内没有文件？

A：由于政策调整或未向您所属类型的用户开放。

##### Q：为什么有些文件（夹）的名称非常奇怪？

A：此类文件为Microsoft提供的机翻。
